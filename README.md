# Tier One Bio

Website for Tier One BioSystems.

Research-grade peptides with published lot-level COAs, 99%+ purity, and US-based fulfillment.

## Development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```

The main application code lives in `site_1.jsx` at the project root. `src/main.jsx` and `src/App.jsx` are minimal wiring files.
